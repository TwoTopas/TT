# Built with The Community Operations Playbook Badge

## Description

A PNG badge (600 × 200 px) with a clean, professional design:

- **Background:** Dark navy (#1a1b2f) to slate gradient
- **Icon:** A stylized gear-and-chat-bubble icon in accent teal (#45e5c8)
- **Text:** "Built with" in light gray (#a0a0b0), "The Community Operations Playbook" in white bold (#ffffff), stacked on two lines
- **Border:** Subtle rounded corners (12px radius) with a thin teal stroke

The badge signals that a community uses structured, professional management systems rather than being run ad-hoc.

## Text to Include

When adding the badge to your server or website, use this HTML/image reference:

```
<img src="https://discordopsplaybook.com/assets/built-with-playbook-badge.png" alt="Built with The Community Operations Playbook" width="300" />
```

Or for Discord server description / about section, use plain text:

```
⚙️ Built with The Community Operations Playbook
```

## Placement Instructions

### Discord Server
- **Server Description (Server Settings → Overview):** Add the badge text at the end of your server description.
- **#welcome or #about channel:** Pin the badge image at the top of the channel.
- **Bot status / rules channel:** Include as a footer note after your community guidelines.

### Website
- **Footer:** Place the badge in your site footer alongside other credentials and certifications.
- **About page:** Add it to your "About the Community" section to build trust with prospective members.
- **Landing page:** Include as a trust signal near your CTA or pricing section.

### Social Media
- **Linktree / Bio link pages:** Add the badge as a visual credential.
- **Email signature:** Include the badge to show prospects your community follows professional standards.

## Why Add It

Displaying the badge tells potential members and partners that your community is run with intention — structured onboarding, tracked engagement metrics, and professional operations. It's a trust signal that separates managed communities from abandoned servers.
