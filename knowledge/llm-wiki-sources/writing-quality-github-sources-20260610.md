## 关键学到的写作规则

### Tier 1 必改词（英文写作中最常见的 AI 痕迹）
delve, landscape, tapestry, realm, paradigm, robust, pivotal, seamless, leverage, cutting-edge, game-changer, utilize, holistic, actionable, impactful

### 翻译规则强化
- 私域 ≠ private domain → owned audience / community-led
- 裂变 ≠ fission → referral program / viral loop
- 红包 ≠ red packet → welcome discount / free trial

### CTA 公式
强: 动作动词 + 具体回报（"Get the Complete Checklist"）
弱: Submit, Sign Up, Learn More
