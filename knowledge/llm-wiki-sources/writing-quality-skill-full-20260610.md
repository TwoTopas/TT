## 三种工作模式

| 模式 | 说明 | 触发词 |
|------|------|--------|
| **detect** | 只检测不修改 | "scan", "detect", "flag only" |
| **rewrite** | 检测+改写（默认） | "rewrite", "clean up", "fix" |
| **edit** | 直接改文件 | "edit in place", "fix file" |

## AI-ism 检测规则

### Tier 1 — 必改词（AI 最常见痕迹）

| 不要用 | 改成 |
|--------|------|
| delve / delve into | explore, look at, dig into |
| landscape (比喻) | field, space, industry |
| tapestry | 直接描述实际复杂度 |
| realm | area, field, domain |
| paradigm | model, approach, framework |
| robust | strong, reliable, solid |
| pivotal | important, key, critical |
| seamless | smooth, easy |
| leverage (动词) | use |
| cutting-edge | latest, newest |
| game-changer | 直接描述改变了什么 |
| utilize | use |
| holistic | complete, full, whole |
| actionable | practical, concrete |
| impactful | effective 或描述实际影响 |
| optimize | improve, make faster |
| innovative | new / 描述具体创新 |

### Tier 2 — 同一段出现 2+ 个就要改

harness, foster, elevate, unleash, bolster, resonate, facilitate, nuanced, ecosystem (比喻), myriad, plethora, encompass, catalyze, cultivate, illuminate, cornerstone, paramount

### 结构性问题

- 句子长度均匀 → 混入短句（3-8词）+ 长句（20+词）
- 段落长度均匀 → 有些段1句话，有些段多句话
- 同义词过度切换 → 同一个词出现3次就是合适的词，保留它
- "let's explore / let's take a look / let's break this down" → 直接说主题
- "In today's X / In an era where" → 直接说事实
- "Moreover / Furthermore / Additionally" → 用 "and, also, on top of that" 或重构
- 三段式强迫症 → 用2个、4个或完整句子
- 空泛结尾 "The future looks bright / Only time will tell" → 删或写具体
- Em dash 使用频率 → 每1000词最多1个
- 粗体过度 → 每个大段最多1处
- "It's worth noting that / It's important to note that / Notably" → 直接说事实

## 转化文案规则

### Headline 公式
- "{实现目标} without {痛点}"
- "The {品类} for {受众}"
- "Never {不愉快事件} again"

### CTA 公式
强: `[动作动词] + [具体回报]` — "Start My Free Trial", "Get the Complete Checklist"
弱: Submit, Sign Up, Learn More, Click Here

### 页面结构
Headline → Subheadline → CTA → Social Proof → Pain → Solution/benefits → How It Works → FAQ → Final CTA

## 中国→英语翻译规则

### 关键术语映射
- 私域 → owned audience / community-led（not "private domain"）
- 裂变 → referral program / viral loop（not "fission"）
- 红包 → welcome discount / free trial（not "red packet"）
- 群托 → community champion / ambassador（not "seeded member"）
- 社群分层 → tiered engagement（not "social stratification"）
- 朋友圈 → content calendar / newsletter（not "moments"）

### 风格规则
- 用 "you" 直接对话
- 引用西方平台：Discord频道、Circle空间、Skool群组
- 禁止中国平台引用（微信、抖音、小红书、知乎）
- 禁止翻译感英语
- 短句优先（英文平均15-18词）
- 读起来必须是母语者写的

## Voice 配置

| Profile | 特点 |
|---------|------|
| casual | 缩略词，短句(≤14词)，第一人称，近零术语 |
| professional | 主动语态，每段一个具体数据 |
| warm | "you"直接称呼，删very/truly/incredibly |
| blunt | 直接说结论，零模糊，短陈述句 |
| technical | X is Y句式，术语在第一次使用时定义 |

---

## 改写 vs 重写决策

| 症状 | 处理 |
|------|------|
| 5+词汇违规 + 3+模式触发 + 句长均匀 | 整段重写 |
| 2-4个词汇违规 | 局部替换 |
| 单个词问题 | 直接替换 |
| 结构上的翻译感 | 用母语者思路重构 |
