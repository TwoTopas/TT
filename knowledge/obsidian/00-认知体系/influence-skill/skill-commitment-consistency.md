# 承诺和一致原理识别与防御

## R — 原文 (Reading)

> "在接受琐碎请求时务必小心谨慎，因为一旦同意了，它就有可能影响我们的自我认知。它不光能提高我们对分量更大的类似请求的顺从度，还能使我们更乐意去做一些跟先前答应的小要求毫不相关的事情"
>
> — 罗伯特·西奥迪尼，第3章 承诺和一致

---

## I — 方法论骨架 (Interpretation)

承诺和一致原理揭示了一个强大的心理机制：一旦人做出承诺（哪怕微不足道），内心和外部都会产生强大压力要求行为与承诺保持一致。其运作有五个关键层面：

1. **承诺触发一致性压力**：一旦选定立场，保持一致的压力逼人觉得必须跟过去的所作所为站在同一阵线。下注后的赌客对所选马匹信心大增——马没变，人变了。
2. **小承诺改变自我形象（登门槛）**：签署一份环保请愿书，两周后就愿意立"小心驾驶"告示牌。小承诺改变了"我是什么样的人"，进而使人遵从一整套与新形象一致的要求。
3. **公开承诺效力最强**：当众选择立场后，人会产生维持它的强烈动机。多伊奇和杰拉德的实验证明，公开记录判断的学生最不愿改变立场。
4. **付出努力增加承诺深度**：忍受严苛入会仪式的成员比轻松入会者更珍视团体。为承诺付出的努力越多，承诺的影响越大。
5. **内心承诺比外部压力更持久**：弗里德曼的儿童实验证明，没有强威胁的孩子（33%违规）比有强威胁的孩子（77%违规）更能长期遵守规则。外部压力一撤就失效，内心承诺能持续。

一致性提供了逃避思考的捷径——按下一致性磁带，就不用再苦苦权衡。这既是效率来源，也是被操控的入口。

## A1 — 书中的应用 (Past Application)

### 案例 1: C国战俘营"以小积大"思想改造
- **问题**: C国需要让A国战俘合作——透露情报、互相揭发、公开谴责自己国家——但不能使用肉体折磨
- **方法论的使用**: C国采用"以小积大"策略：从温和陈述（"A国并不完美"）→列问题清单并签名→小组讨论→写文章扩充→在广播中播报姓名和文章。每个小步骤都是一个承诺，推动战俘调整自我形象以与行为保持一致
- **结论**: 战俘因写下、签名、公开这些渐进承诺，被迫调整自我认知成为"合作者"
- **结果**: 据沙因研究，几乎所有战俘都以某种方式与C国合作过，战俘的政治态度发生了根本性转变

### 案例 2: 沙滩偷窃实验与玩具公司操控家长
- **问题**: 口头承诺的力量有多大？玩具公司如何利用家长对孩子的承诺？
- **方法论的使用**: 沙滩实验中，请受试者"帮忙看着东西"的简单口头承诺使干预率从20%飙升至95%。玩具公司在圣诞节前广告某款玩具让孩子缠着父母承诺购买，故意不供货，节后再打广告触发"你答应过的"压力
- **结论**: 即使随意的口头承诺也能激活强大的一致性驱力；家长即使识破策略也无法脱身——不想在孩子面前食言
- **结果**: 玩具公司每年在同一个家庭身上赚取双倍利润

### 案例 3: 弗里德曼与弗雷泽"安全驾驶"告示牌实验
- **问题**: 微不足道的小承诺能否影响完全不同领域的重大决策？
- **方法论的使用**: 先让业主签署"保护加州美丽环境"请愿书（几乎人人都签），两周后请他们立大型"小心驾驶"告示牌。虽然环保和安全驾驶内容无关，但签署请愿书改变了人的自我认知（"我是有公益精神的好市民"）
- **结论**: 无关领域的小承诺使大请求顺从率从17%提高到约50%，证实承诺通过改变自我形象产生跨领域影响
- **结果**: 证明了登门槛策略的核心机制：不是内容关联，而是自我形象的改写

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **因之前的小承诺感到不得不做更大的事**：如签了环保请愿书后被要求参加更多活动，或对销售员说了"我对这个有兴趣"后觉得必须购买
2. **明知方向有误却因"说到做到"而无法退出**：如在亏损的投资上继续追加（沉没成本），或在一个不合适的岗位上因公开承诺而硬撑
3. **被要求做微不足道的表态或签名**：如电话募捐员先问"今晚感觉如何"再请求捐款，或请愿书签名、调查问卷中的立场表态

### 语言信号 (用户的话里出现这些就应激活)

- "我已经答应了/说了要做，不好反悔"
- "既然开始了就不能半途而废"
- "我已经投了这么多进去，现在退出来就全白费了"

### 与相邻 skill 的区分

- 与 `lowball` 的区别: 抛低球侧重"承诺后撤回最初诱因"的手法，承诺和一致原理侧重"承诺如何改变自我认知并驱动后续行为"的底层机制——抛低球是承诺一致性原理的一种应用
- 与 `reciprocation` 的区别: 互惠原理的驱动力是"亏欠感"，承诺和一致的驱动力是"一致性压力"——前者是"我欠他的"，后者是"我必须跟自己之前的立场一致"
- 与 `rejection-then-retreat` 的区别: 拒绝-后撤是通过"让步"触发义务感，而承诺和一致是通过"让步后的协议"产生承诺感——拒绝-后撤的副作用之一就是产生承诺

## E — 可执行步骤 (Execution)

当 skill 被激活后, agent 应按以下步骤执行:

1. **识别当前行为是否被之前的承诺驱动**
   - 完成标准: 确认是否存在一个先前的承诺（口头、书面、公开的表态）正在施加一致性压力

2. **评估承诺的性质**
   - 完成标准: 判断该承诺是(a)经过深思熟虑的真实承诺，还是(b)被诱导的/微不足道的/无意中做出的小承诺
   - 判停条件: 若是经过深思熟虑的真实承诺，则一致性压力是合理的——跳到步骤4

3. **分离一致性压力和实际利弊**
   - 完成标准: 暂时忘掉"我之前答应了/说了要做"，仅凭当前信息独立评估"如果现在重新选择，我会怎么做？"
   - 核心问题: "知道了我现在掌握的情况，要是时间能倒流，我还会做同样的选择吗？"——捕捉第一瞬间的直觉反应

4. **区分理性坚持与机械一致**
   - 完成标准: 判断坚持的动机是基于新发现的合理理由，还是仅为逃避思考的"死脑筋"一致性
   - 判断信号: 如果肠胃发出警铃（感到自己中了套），就是机械一致

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **正常信守承诺的行为**：承诺和一致原理在社会运作中不可或缺——如果承诺是经过深思熟虑且自愿做出的，保持一致是正确的行为
- **专业职责和契约义务**：医生对病人的承诺、律师对客户的承诺——这些是职业伦理，不是一致性陷阱
- **日常习惯的正面坚持**：如坚持锻炼、坚持学习——这些是承诺一致性的正面应用

### 作者在书中警告的失败模式

- **"死脑筋地保持一致"（ce05）**：一致性磁带让人逃避思考，躲进"死脑筋的城堡"。超自然冥想讲座中，人们听到逻辑反驳后反而更急切地掏钱报名——因为反驳太有说服力，他们需要在理性占上风之前用快速承诺筑起对抗理智的城墙
- **登门槛效应与自我形象的被动改写（ce06）**：微不足道的小承诺改变自我认知，使人答应完全不同但性质相符的大请求。电话募捐员先问"今晚您感觉如何"再请求捐款，成功率翻倍

### 作者的盲点 / 时代局限

- 六大原则主要基于20世纪60-80年代美国的实验研究，社交媒体时代的公开承诺（如朋友圈打卡、公开立flag）可能产生更复杂的效果
- 对"机械一致"的防御建议（倾听肠胃/心灵深处的信号）依赖个人自我觉察能力，实际操作性有限
- 文化差异：集体主义文化中，公开承诺的压力可能比个体主义文化更强

### 容易混淆的邻近方法论

- **沉没成本谬误**：与承诺一致性密切相关，但沉没成本侧重"已经投入的资源"驱动继续投入，承诺一致性侧重"已经做出的表态"驱动继续坚持——两者常同时出现但机制不同
- **习惯形成**：一致性压力可以帮助建立好习惯，这本身是正面的——不应将所有一致性都视为需要"防御"的操控
