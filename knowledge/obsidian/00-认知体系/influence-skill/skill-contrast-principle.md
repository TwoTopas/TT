# 对比原理识别与应用

## R — 原文 (Reading)

> "基于先前所发生事件的性质，相同的东西会显得极为不同。"
>
> — 西奥迪尼，第1章 影响力的武器

---

## I — 方法论骨架 (Interpretation)

对比原理的核心发现：两样东西一前一后展示时，我们感知到的差异会比实际差异更大。同一件事物，因为前置参照物不同，会产生截然不同的主观感受。

这不是普通的"比较"，而是一种系统性的感知扭曲：
1. **先后顺序决定感知**: 先接触的事物会成为后接触事物的"锚点"，后续感知被拉离这个锚点。
2. **扭曲是无意识发生的**: 你无法通过"我知道这是对比原理"来完全消除它的效果，就像你知道手放进过热水，放进常温水时仍然觉得凉一样。
3. **商业中无处不在**: 先卖西服再卖配饰（配饰显得便宜）、先看烂房再看目标房（目标房显得极好）、先报高价再报低价（低价显得合理）。

识别对比原理运作的关键信号：当你对某事物的感受在短时间内发生了剧烈变化，而这个变化恰好发生在你接触了一个极端参照物之后。

## A1 — 书中的应用 (Past Application)

### 案例 1: 房地产"垫底货"策略
- **问题**: 如何让普通房源显得更有吸引力、促进成交？
- **方法论的使用**: 房地产销售员菲尔每次带客户看房时，先看几套破旧且标价虚高的"垫底货"。垫底货不打算卖，只是作为对比参照物。看完垫底货后，目标房源在对比下显得远比实际更优越。
- **结论**: 对比原理使得真正目标房源在客户眼中"眼睛一亮"，产生远超客观品质的吸引力。
- **结果**: 通过先展示劣质选项制造对比效果，显著提高了目标房源的吸引力和成交率。

### 案例 2: 德瑞贝克兄弟服装店对比定价骗局
- **问题**: 如何让顾客心甘情愿按高于实际的价格购买服装？
- **方法论的使用**: 西德假装耳背，问价时后面兄弟哈里报真实价42美元，西德假装听错告诉客户"22美元"。客户在"42美元"和"22美元"的对比下，觉得22美元是天大的便宜。
- **结论**: 虚假的锚定价格（42美元）使实际价格显得极低，客户抢着在"错误"被发现前成交。
- **结果**: 许多客户以为自己占了便宜而快速成交，实际上被对比原理操控。

### 案例 3: 汽车经销商配件加价策略
- **问题**: 为什么汽车销售员先卖车再加装配件，而不是打包报价？
- **方法论的使用**: 先确定车价（大额，如2万美元），再逐一提出配件（几百美元）。配件价格在大额车价的对比下显得微不足道。如果打包报价，消费者会仔细审视总价。
- **结论**: 对比原理让小额支出在大额支出后显得可以忽略，消费者不知不觉中接受了大量附加费用。
- **结果**: 配件价格加起来让整车价格"注水"般涨上去，顾客目瞪口呆却已签好合同。

## A2 — 触发场景 (Future Trigger) ★

### 用户会在什么情境下需要这个 skill?

1. **购物或谈判中被引导感受扭曲**: 用户在购买或谈判过程中，对价格的感受似乎被某种"顺序"操控了（如先被带看高端型号再看中端型号就觉得中端便宜）。
2. **事后发现决策失误**: 用户事后反思时发现，自己当初觉得"划算"或"合理"的东西，脱离了当时的情境后发现并不划算。
3. **设计展示或报价策略**: 用户需要向他人展示方案或报价，想知道如何利用先后顺序优化说服效果（或如何避免被对方利用）。

### 语言信号 (用户的话里出现这些就应激活)

- "先看了那个之后，这个突然觉得挺便宜的"
- "相比之下这个就不那么贵了"
- "跟前面那个比起来这个简直太好了"
- "我也不知道为什么就觉得划算"

### 与相邻 skill 的区分

- 与 `click-whirr` 的区别: 对比原理是 Click-Whirr 的一种具体触发方式——"先后顺序的对比"作为触发特征。但对比原理更侧重于感知扭曲的机制（锚定效应），而非通用的自动反应框架。
- 与 `rejection-then-retreat` (s04) 的区别: 拒绝-后撤策略同时调用了互惠原理和对比原理，对比原理只是其中的一个子机制。如果用户只关注"先后呈现如何扭曲感知"，用对比原理；如果用户关注"对方让步后我感到有义务回报"，用互惠或拒绝-后撤。

## E — 可执行步骤 (Execution)

当 skill 被激活后， agent 应按以下步骤执行:

1. **识别锚点和当前感知**
   - 完成标准: 明确指出用户当前的感受是什么，以及产生这个感受的"前置参照物"（锚点）是什么。

2. **评估感知扭曲程度**
   - 完成标准: 将当前事物从锚点影响中剥离出来，独立评估其客观属性（实际价格、实际质量、实际吸引力），并与锚定后的主观感受做对比，量化扭曲程度。

3. **判断是否被人为操控**
   - 完成标准: 判断锚点是自然产生的还是被对方故意设置的（如销售员故意先展示高价商品）。如果是人为设置，指出操控意图。

4. **制定应对策略**
   - 完成标准: 如果是防御场景，提供"重置锚点"的方法（如先做独立评估、引入第三方参照、在决策前暂停）；如果是应用场景，提供如何合法使用对比原理优化展示顺序的建议。

## B — 边界 (Boundary) ★

### 不要在以下情况使用此 skill

- **同时并列比较**: 用户同时看到多个选项并排比较时，对比原理的作用有限——它的核心是"先后顺序"产生的感知扭曲，不是"同时对比"。
- **专业知识充分的领域**: 当用户对某领域有足够专业知识，能够独立判断事物的客观价值时，对比原理的效果会大幅减弱。

### 作者在书中警告的失败模式

- **德瑞贝克兄弟骗局中的系统性操纵（c04）**: 客户被虚假锚定价格操控，以为自己占了便宜而快速成交。这是对比原理被恶意利用的典型——锚点（42美元）是故意制造的虚假参照。
- **水门事件中的对比+互惠双重操控（c12）**: 利迪先提100万方案再退到25万，25万在前两个极端方案的对比下显得"可以接受"。对比原理使25万的愚蠢程度被系统性低估。

### 作者的盲点 / 时代局限

- 书中主要关注线下场景（服装店、房地产、汽车销售），未涉及线上环境中更隐蔽的对比操控：如电商"划线价"（虚构原价制造折扣感）、酒店预订网站"只剩2间"的稀缺对比、社交媒体上的精修照片对自我评价的扭曲等。
- 对比原理与行为经济学中的"锚定效应"高度重叠，但作者未与卡尼曼等人的研究做交叉引用，可能导致读者低估该原理的学术深度。

### 容易混淆的邻近方法论

- **"锚定效应"与"对比原理"**: 锚定效应是认知偏差的一种，指第一个接收的数字会影响后续判断；对比原理是更广泛的感知现象，不仅限于数字。两者的区别在于：锚定主要影响数量判断，对比原理影响所有感官知觉（重量、温度、吸引力等）。
- **"普通比较"与"对比原理扭曲"**: 正常的比较是同时获得多个信息后做理性分析；对比原理的扭曲是先后顺序造成的无意识感知偏差——同一杯水，先摸热水就觉得凉，先摸冷水就觉得热。
