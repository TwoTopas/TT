# 行为经济学（怪诞行为学）— 实操 SKILL

## 核心框架

### 核心原则（来自 Dan Ariely《Predictably Irrational》）

| 原则 | 一句话 | 对 SepticSaver 的应用 |
|------|--------|---------------------|
| **免费的力量** | 人们会非理性地偏好免费的东西，即使总成本更高。免费降低决策门槛但也降低了感知价值。 | 45天免费试用已实现。注意不要永久免费，否则用户不付费。 |
| **所有权效应** | 一旦用户觉得"这是我的"，估价比实际价值高2-3倍。45天试用后用户已经"拥有"了数据。 | 试用期结束后数据只读但不删除 — 用户不想失去已有的数据。 |
| **社会规范 vs 市场规范** | 社会规范（朋友之间帮忙）和市场规范（付钱买服务）不能混用。"打赏"混淆了这两种规范。 | ✅ 我们的模式是明确的市场规范— 一次性付费，没有混淆。 |
| **选择过载** | 选项越多，决策越难。3个选项是上限。$80/$150两级定价合理。 | ✅ 免费 / $80 / $150 三个选项刚好。不要加更多。 |
| **锚定效应** | 第一个价格决定后面的感知。先让用户到 Tank Track $399/月 → $150 买断显得便宜。 | ✅ 弹窗有竞品参照。但免费版用户没有锚定——他们只看到"免费"。 |
| **预期效应** | 用户使用产品的体验会被他们的预期影响。告诉他们"这个软件很好用" → 他们会觉得更好用。 | 欢迎页的文案需要建立正面预期。 |
| **诚实因素** | 提醒人们诚实标准可以减少欺骗行为。 | 对订阅用户告知"我们只记你真实的服务数"，建立信任。 |
| **拖延症** | 人们会无限期推迟非紧急决策。免费试用到期的用户会拖着不决定。 | 试用期最后 7 天倒计时已实现。但还需要在最后 24 小时发邮件。 |
| **分块支付 vs 一次性支付** | 一次性支付痛感强但一次过。分期支付痛感低但总和更高。 | $150 一次性 — 痛感中等。$80/basic — 痛感低。合理。 |

### 对 SepticSaver 的关键洞察

**1. 所有权效应是你最强的武器。**
- 用户免费试用 45 天，导入了 200 个客户
- 这段时间里，在这套系统里标记了几个客户
- 45 天结束时，用户已经心理上"拥有"了系统里的数据和组织方式
- 要失去它 → 损失厌恶驱动付费

**2. 免费的力量是把双刃剑。**
- 免费试用增加注册
- 但如果永久免费版功能够用 → 用户不会付费
- 所以 50 客户限制是必要的——让用户觉得不够用
- ⚠️ 但 50 个限制不要影响初始体验——用户注册时要能导入所有客户看到冲击，但第 30 天开始提醒"你已经 48/50 了"

**3. 社会规范 vs 市场规范的边界必须清晰。**
- 创始人计划有专门的"VIP"身份—这是社会规范（归属感）
- 付费本身是市场规范（交易）
- 两者不冲突——用户"买"了创始会员身份，属于正义市场规范中的身份认同

**4. 拖延症是最大的转化杀手。**
- 用户 99% 会在试用期最后 1 天决定
- 到那时可能因为太忙或忘了而错过
- 必须在最后 7 天、3 天、1 天做三次触达
