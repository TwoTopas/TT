# 美国SaaS — 境外未知风险清单

> 你可能不知道的风险，不是中国的，是美国的
> 日期：2026-06-07

---

## 🔴 高风险（不知道就会踩坑）

### 1️⃣ IRS Form 5472 — 漏报罚款 $25,000

**这是非居民美国LLC所有人最大的隐藏陷阱。**

| 项目 | 内容 |
|------|------|
| 什么情况需要报？ | **只要你是非美国居民，有美国LLC，就必须报** |
| 报什么？ | IRS Form 5472 + 模拟Form 1120 |
| 截止日期 | **每年4月15日**（可延期到10月15日） |
| 罚款 | **$25,000起步**，每迟一个月再加 |
| 收入$0也要报吗？ | **是的，一分钱没赚也要报** |
| 找谁报？ | 专业跨境税务师（~$300-500/年） |

**这条是真实风险。** 很多中国开发者不知道，LLC放着不管第二年就收到$25K罚单。

### 2️⃣ Stripe 账户被冻结

Stripe 对非居民账户的审核非常严格：

| 触发因素 | 风险 |
|---------|------|
| 中国IP登录 | 触发风控审核 |
| 短时间内交易激增 | 要求提供证明文件 |
| 退款率>1% | 可能冻结账户 |
| 账户长时间闲置后突然活跃 | 触发KYC复审 |

**解决方案：** 始终用美国节点VPN登录Stripe；不要用中国IP直接访问。

### 3️⃣ Mercury 银行关户

| 风险 | 说明 |
|------|------|
| 长期不登录 | 可能判定为僵尸账户 |
| 频繁大额进出 | 触发反洗钱审核 |
| 关联账户出问题 | 你提到的其他被关户案例 |

**Mercury 不是FDIC保险银行**（它是Choice Financial Group的合作伙伴）。账户里的钱理论上是有保障的，但关户后资金提取可能需要几周。

## 🟡 中等风险（需要注意）

### 4️⃣ TCPA / SMS 合规

美国对商业短信监管极严：

| 要求 | 罚款 |
|------|------|
| 必须获得客户书面同意接收短信 | 每条违规 SMS 最高 $1,500 |
| 必须提供退订方式 | 同上 |
| 10DLC 注册（2026年起强制执行） | 未注册的短信可能被运营商拦截 |

**SepticSaver 已经有 sms_consent 功能 + 退订机制 + TOS 免责条款。** 这部分你已经比别人准备得好。

### 5️⃣ 州年报 & 特许经营税

| 事项 | 费用 | 错过后果 |
|------|------|---------|
| Wyoming 年报 | $50/年 | LLC被注销 |
| 注册代理费 | $50-100/年 | LLC失去合法地址 |
| Delaware 特许经营税 | $300/年（如果选DE） | LLC被注销 |

**建议：** 注册 Wyoming 或 New Mexico（年报成本最低）

### 6️⃣ 数据泄露责任

| 场景 | 风险 |
|------|------|
| 数据库被黑，客户信息泄露 | 可能被集体诉讼 |
| 没有 HTTPS | 浏览器标记"不安全"，客户流失 |
| 没有备份 | 数据丢失 = 业务停止 |

## 🟢 低风险（但听说过）

### 7️⃣ 被起诉

$29-49/月的工具，被起诉的概率极低。但：
- TOS里写好"责任上限为12个月服务费"（SepticSaver已有 ✅）
- LLC结构保护个人资产（不要用个人账户收款）

### 8️⃣ 竞品收购/威胁

美国竞品不能把你怎么样。只要你没有侵犯专利/商标，他们拿你没办法。

### 9️⃣ 入境美国被问

| 场景 | 风险 |
|------|------|
| 你去美国旅游，海关问"你在美国有公司吗" | 🟢 如实回答：LLC，100%远程运营，没有美国员工 |

## 📋 风险控制清单

| 风险 | 怎么控制 | 成本 |
|------|---------|------|
| Form 5472罚款 | 每年找税务师报税（$300-500） | $300-500/年 |
| Stripe被冻结 | 用美国VPN + 低退款率 + 透明业务描述 | VPN $0-5/月 |
| Mercury关户 | 保持活跃，每月至少登录一次 | $0 |
| SMS罚款 | 已有 sms_consent + 退订功能 | ✅ 已就绪 |
| 州年报过期 | 设日历提醒（每年4月前） | $0 |
| 数据泄露 | HTTPS + 加密 + 定期备份 | $0-10/月 |
| 被起诉 | TOS写好免责条款 | ✅ 已就绪 |

### 每年固定成本

```
LLC年报      $50-100
注册代理      $50-100
税务师报税    $300-500（Form 5472）
合计         ~$400-700/年
```

> **这部分是很多中国开发者不知道的隐藏成本。** 即使你的 SaaS 一分钱没赚，每年也要花 $400-700 维持合规。
