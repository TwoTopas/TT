# QuoteIQ— 烟囱行业垂直软件架构分析

> 来源：Reddit r/smallbusiness 推广帖 + QuoteIQ 官网描述（官网屏蔽非浏览器访问）  
> 时间：2026-05 分析  
> 用途：调研竞品架构，为化粪池行业 SaaS 设计做参考

## 一句话定位

**轻量 CRM + 现场服务管理（FSM）混合体。** 不是 ServiceTitan 级别的重型系统，也不是纯 CRM。走低价量大路线，标榜 "over 40,000 contractors trust QuoteIQ"。

## 目标行业（多行业垂直，非单一）
QuoteIQ 广告覆盖 8+ 个垂直行业（从推广帖推销词推断）：
- Chimney Sweep（烟囱清洁）
- Pest Control（害虫控制）
- HVAC
- 以及其他服务行业

它像一个 CRM 套壳 + 行业适配。不是为烟囱行业深度定制的产品，而是在同一套 CRM 上给每个行业做了一些预设模板。

## 功能矩阵（基于推广帖内容还原）

| 功能模块 | 包含 | 行业深度 |
|----------|------|---------|
| **排期与调度** | 月度日历视图，按服务区域和 CSIA 认证级别分配 | 通用 CRM 级别 |
| **现场报价与签署** | 现场生成报价、电子签名、访问客户历史 | 通用 CRM 级别 |
| **支付处理** | 估价转发票、现场刷卡、QuickBooks 集成 | 通用支付集成 |
| **客户通信** | 自动提醒、季节性通知、安全提示、评论请求 | 通用 CRM 级别 |
| **库存跟踪** | 烟囱刷具/杆子/防风帽/阻尼器等耗材跟踪 | 行业专用（烟囱） |
| **员工管理** | CSIA 认证跟踪、继续教育要求 | 行业专用（烟囱） |
| **营销辅助** | "Virtual Call Team" 捕捉未接来电、"Budget Built Sites" SEO | 行业专用（烟囱文案） |
| **报告** | 基础报告，非深度分析 | 通用 CRM 级别 |

## 定价策略
- 帖子未提供具体数字
- 40,000+ 用户 → 走低价量大路线
- 估计 $50-150/月（对比 ServiceTitan $500+/月/用户）
- 对比：同领域的 Jobber 起步 $69/月，Buildertrend $99-300/月
- 大概率是 $99-199/月 档位

## 架构特点

```
QuoteIQ = 多租户 SaaS + 行业适配层
├── 核心框架（通用 CRM + 排期 + 支付）
│   ├── 排期引擎（统一）
│   ├── 报价/发票引擎（统一）
│   ├── 支付管道（统一，QuickBooks 集成）
│   └── 客户数据库（统一）
├── 行业适配层（预设）
│   ├── Chimney 预设：CSIA 认证类型、刷具/烟囱库存
│   ├── Pest 预设：化学品跟踪
│   ├── HVAC 预设：制冷剂跟踪
│   └── 行业特定 SEO 文案
└── 基础设施
    ├── Virtual Call Team（第三方集成？客服外包？）
    ├── Budget Built Sites（网站搭建服务）
    └── Google Review 自动请求
```

## 关键劣势

### 1. 不是行业深度产品
QuoteIQ 的核心是 CRM。它没有化粪池行业的特定功能：
- 3-5 年维护周期自动计算和提醒
- 手绘地图数字化和存储
- 县公立维护记录集成
- 处置费用和路线跟踪
- 数字化的客户档案卡

### 2. 护城河是品牌，不是技术
- 40,000+ 用户是硬数字
- 但它的功能任何人都可以复刻
- 技术壁垒 = 低

### 3. 价格空位
```
ServiceTitan ($500+/月) → 太重，夫妻店用不起
     ↑ 你的机会 ($99/月)
QuoteIQ/Jonber ($69-199/月) → 烟囱行业有，化粪池行业没有
     ↑ 你的另一个机会
Spreadsheets + 纸笔 ($0) → 现状，但效率低
```

## 对化粪池 SaaS 的启示

### QuoteIQ 证明了什么
1. **垂直行业专用 CRM 有市场** — 40,000+ 用户是验证
2. **烟囱行业可以做，化粪池行业也可以做** — 需求结构类似
3. **QuoteIQ 的模式是"通用 CRM + 行业预设"** — 不是深度行业产品
4. QuoteIQ 不做的东西 = 你的差异化点

### QuoteIQ 没做什么
1. 不做数据基础设施（县公开记录）
2. 不做行业数据飞轮（客户数据积累）
3. 不做 AI 功能（自动预测、报告生成）
4. 不做 PE 整合方管理
5. 不做引导夫妻店从"数据"到"软件"的 GTM 路径

### QuoteIQ 在化粪池行业的情况
- 帖子中**没有任何化粪池从业者提到 QuoteIQ**
- 化粪池从业者提的是：HouseCall Pro、ServiceCore、SeptiTrack、Tank Track
- **QuoteIQ 在化粪池行业的品牌认知 = 0**
- 但 QuoteIQ 如果盯上化粪池行业，可以快速适配（它的架构是"通用 CRM + 行业预设"）

## 竞争策略（如果做化粪池 SaaS）

```
QuoteIQ → 通用 CRM + 行业预设，轻量
ServiceTitan → 重型 FSM，太贵
SeptiTrack/Tank Track → 化粪池专用但产品糙
SepticSnap → 只做检测报告，不做管理
```

**你的机会：** 不做 CRM，做 **化粪池行业的数据基础设施 + 排期工具**。
- 不是"又一家 CRM"
- 是"拿着数据来找你，从数据演进到排期"
- 这个路径 QuoteIQ 走不通，因为它的 GTM 是"买我的软件"，不是"我有你的客户名单"
