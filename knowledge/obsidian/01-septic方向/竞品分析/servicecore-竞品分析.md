# ServiceCore 竞品分析（2026-06-03）

> 来源：servicecore.com/features/ + servicecore.com/pricing/ + servicecore.com/plans/ + servicecore.com/septic-business-software/
> ServiceCore 是**便携卫生间+化粪池+垃圾桶**行业的综合运营软件

---

## 关于 ServiceCore 作为化粪池软件的纠正

**之前错误认知：** ServiceCore 只适用于多卡车中型运营商，不适合夫妻店。
**纠正：** ServiceCore 官网明确说"Whether you run one truck or an entire fleet"和"1 to 3 trucks"是他们的目标客户群之一（表单中有 1-3 车的选项）。**一车也能用，但$200+/月的价格对一车店仍然太重。**

## 化粪池专用页面（servicecore.com/septic-business-software/）

### 针对化粪池的核心功能宣称

| 功能 | 说明 |
|------|------|
| 服务类型分类 | pumping, installation, repair, mortgage inspection 等 |
| 现场信息记录 | 安装日期、入口位置、挖掘要求、软管长度、risers、罐体材质、泵出量、下次建议维护日期 |
| 手机端导航 | 逐向导航到工单 |
| 即时回传 | 司机完成服务后数据立即传回办公室 |
| 路由优化 | 多司机时一键创建和优化路由 |
| 现场收款 | 现金/支票/信用卡（ServiceCorePay 费率比 Stripe/PayPal 低） |
| 合规报告 | 一键生成：月度工单、泵出加仑数、按处置站点/邮编/城市统计的处置量 |
| 检测报告 | 从现场通过邮件发送检测报告 |
| **28天计费周期** | 明确支持油脂拦截池的 28 天计费 |

### "一车也能用"的证据

表单中 `# of Trucks (Range)` 选项包括 **"1 to 3"**，说明 ServiceCore 的最低目标用户是**1车起**。

### "Keep What Works. Simplify The Rest."

这是他们的标语——说明他们也意识到化粪池商家不需要颠覆现有流程，只需要简化。

## 对 SepticSaver 的影响

1. **ServiceCore 对夫妻店的入门门槛降低到了1车**，但价格门槛不变（未公开，估$200+/月）。$49 的 SepticSaver 仍然便宜 4 倍。

2. **ServiceCore 明确支持 28天计费周期——这是油脂拦截池的杀手功能。** 如果 SepticSaver 未来要做油脂业务线的提醒，这个功能要考虑到。

3. **ServiceCore 支持"mortgage inspection"** 作为服务类型，说明化粪池检查也是高频需求。

4. **"No more notecards!"** 和 **"replace the stack of spreadsheets, calendars, and sticky notes"**——ServiceCore 也把"纸笔Excel"作为获客切入点，和 SepticSaver 的定位一致。

## 核心功能详解（从features页面提取）

### 客户和工单管理
- 在一个界面添加新客户和工单（几点击）
- 按名称/公司/地址/电话/邮件搜索
- 每个客户的所有工单和租赁快速概览
- 创建客户类型并设定自定义定价
- 跟踪所有客户站点、信用卡信息、联系人和更多

### 账单和应收
- **批量给数百客户开票**（一键）
- 行业标准计费周期
- 可自定义租金率和定价
- 可添加附加费
- 支持在线付款
- 邮件发送或打印发票
- 自动**查看所有未结余额**（无需手动查找）

### 排程、路由和调度
- **拖拽调度**——调整路由和分配工单
- **一键路由优化**
- 日/周视图或地图视图
- 按司机、工单或租赁类型、部门筛选
- 手动调度或使用**系统建议**（基于最近位置、总里程、工单数）
- 查看每条路由的服务时间、驾驶时间和总时间

### 手机App
- 预览整个路由或所有工单的每个细节（一个屏幕）
- iOS和Android逐向导航
- 一键记录实际服务时间
- 标记完成/部分完成/无法完成
- 添加备注和照片
- 标记额外费用
- 捕获客户签名

### 库存管理
- 在**地图上看到每个单元的确切位置**
- 日历查看可用性，信心十足地排单
- 消除超订或低预订
- 多产品线——在一个屏幕上看所有库存
- 实时知道是否有损坏并添加损坏/清洁费

### 报告
- 运营概览
- 按工单类型/服务地点/服务区域的报表
- 日/月工单报表
- 账龄报告（所有未付余额）
- 按客户/发票行/收入维度的销售报表

### 数字营销（附加服务）
- 建站+管理
- 增加Google评论
- 让潜在客户在线找到你
- 社交媒体增加曝光
- **每月进度报告**显示广告效果

## 对 SepticSaver 的战略启示

1. **ServiceCore 和 SepticSaver 没有竞争关系。** 它是给"多卡车"公司的重型工具，夫妻店不会花$200+/月+培训时间去用它。

2. **ServiceCore 不做化粪池专用功能。** 它的化粪池功能是"便携卫生间+垃圾桶+化粪池"通用版，没有3-5年排期周期、化粪池系统类型、罐体规格记录等垂直功能。

3. **但不公开定价是所有中型以上竞品的通病。** SepticSaver 的透明定价（$49/月）本身就是差异化武器。

4. **ServiceCore 的存在验证了一个事：** 在化粪池+便携卫生间+垃圾桶这个"脏行业"领域，已经有软件公司在赚钱（556K tracked units, 3.5M completed jobs, $400M revenue processed）。市场是真实的。

5. **夫妻店到ServiceCore之间有个空位：** 夫妻店用Excel/笔记本 → 需要的东西是什么？是 ServiceCore 的 1/10 功能但 $49/月 的中间件。**这就是 SepticSaver 的位置。**
