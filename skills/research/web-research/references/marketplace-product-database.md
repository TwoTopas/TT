## Key Insights

1. **Software & Plugins is the most active subcategory for small tools** — 24+ products with ratings visible
2. **Mac desktop tools can sell well**: Supercharge ($20, 254 ratings) proves a simple, well-designed Mac utility can generate significant sales without an existing audience
3. **Creative tool plugins command premium prices**: Retro Diffusion ($20, 228 ratings), TimeOffset ($17+, 45 ratings)
4. **Chrome extensions are viable**: Screen Ruler ($29), CRAFTR ($99), Meet Enhancement Suite ($4/mo) — varied pricing models work
5. **"Tiny tools" with a single specific function work**: Default Browser ($6, 10 ratings), Hi-Res Apple Music ($2, 17 ratings) — solve exactly one problem
6. **Template/kits have 0 sales** in this space: products like "Freelancer Client Onboarding Kit" priced at $6-29 consistently show 0 ratings
7. **Sindre Sorhus is the benchmark**: One developer with multiple Mac tools (Supercharge, Default Browser, Power User Bundle) all selling well — proves the solo dev + Gumroad model
