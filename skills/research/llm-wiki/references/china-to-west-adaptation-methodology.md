# China-to-West Digital Product Adaptation Methodology

Use this when adapting Chinese digital products (templates, guides, SOPs, tools) for English-speaking markets.

## Step 1: Platform Mapping

Map Chinese platform concepts to correct Western equivalents BEFORE writing anything:

| Chinese | Western | Key Difference |
|---------|---------|----------------|
| 微信群 | **Discord / Circle / Skool** | Channel-based vs. chat-based |
| 抖音 | **TikTok** | Same product, different regional instance |
| B站 | **YouTube** | B站 has stronger community culture (弹幕) |
| 小红书 | **Instagram + Pinterest mix** | No exact equivalent |
| 公众号 | **Substack / Newsletter** | Content subscription model |
| 知乎 | **Reddit / Quora** | Topic-driven Q&A |
| 企业微信 | **Slack** | Channel-first vs. chat-first |
| 微博 | **X/Twitter** | Microblogging |

## Step 2: Terminology Translation

| Chinese | DO NOT Use | Use Instead |
|---------|------------|-------------|
| 私域/私域流量 | private domain | Owned audience / community-led / first-party traffic |
| 裂变 | fission | Referral program / viral loop / member-get-member |
| 社群分层 | group tiering | Member tiers / segmentation / engagement scoring |
| 拉新/激活/留存/转化 | — | Acquisition / Onboarding / Retention / Monetization |
| 群托 | seeded member | Community champion / ambassador (transparently recruited) |
| 红包 | red packet | Welcome discount / free trial / content upgrade |
| 人设/IP | persona | Brand voice / community manager identity |
| 复盘 | — | Retrospective / post-mortem |
| 话术 | — | Script / copy / messaging |
| KOL/KOC | key opinion leader | Creator / influencer / advocate |

## Step 3: Structural Adaptation

Chinese and Western community operations follow fundamentally different rhythms:

| Chinese Model | Western Model |
|---------------|---------------|
| **Hourly schedule** (post every 2-3 hours) | **Weekly rhythm** (daily themes, automated sequences) |
| **Chat-first** (WeChat group as primary) | **Channel-first** (topic-based channels) |
| **Relationship-driven** (先拉群, 再给内容) | **Content-driven** (先看内容, 再加入) |
| **Personal account ops** (1-on-1 DMs) | **Automated onboarding** (bot sequences + email) |
| **Group seeding** (hired plants) | **Ambassador programs** (transparent volunteers) |

## Step 4: Forbidden Terms (Gate Check)

Before delivering any output, search for and remove:
- WeChat, 微信, WeChat groups, 私域, 小红书, 抖音, B站, 知乎, 微博
- Red packet, hongbao, QR code (as growth tactic)
- Guanxi, mianzi (Chinese business terms)
- Any reference to Chinese platforms as examples

## Step 5: Style — Must Sound Native

- Use "you" (direct address)
- Short sentences (15-20 words average)
- Contractions (you'll, don't, it's)
- Platform-specific: mention Discord channels, Circle spaces, Skool groups
- No corporate jargon (leverage, synergistic, holistic)
- No translation-ese ("it is worth noting", "as shown below")
- No passive voice where active works better
