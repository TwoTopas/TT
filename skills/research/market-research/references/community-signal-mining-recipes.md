# Community Signal Mining — Search Recipes

Verified working DuckDuckGo site-search queries for business/market validation.

## Septic / SepticSaaS

```bash
# Reddit: software needs
ddgs text -q "site:reddit.com septic tank scheduling software" -m 5
ddgs text -q "site:reddit.com septic service management software" -m 5

# Reddit: business economics
ddgs text -q "site:reddit.com/r/sweatystartup septic pumping" -m 5
ddgs text -q "site:reddit.com/r/smallbusiness septic" -m 5

# Quora: industry Q&A
ddgs text -q "site:quora.com septic tank business profit" -m 5
ddgs text -q "site:quora.com septic pumping cost" -m 5

# Facebook: local providers
ddgs text -q "site:facebook.com septic pumping service" -m 5
```
