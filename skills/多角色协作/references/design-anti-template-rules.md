# 反模板设计规则（喂给Claude Code用）
# 来源: 2026-06-23 session - 用户说"太模板化了"

## 绝对不能出现的设计
- ❌ 纯白卡片在浅灰背景上一字排开
- ❌ 系统默认字体
- ❌ 单一的#007aff蓝色强调色全篇都用
- ❌ 卡片等宽等距平均分布
- ❌ 所有元素对齐方式一样

## 必须实现的设计
- ✅ 暖色背景 #f8f7f4 替代纯灰 #f2f2f7
- ✅ 卡片暖白 #fefefe 替代纯白 #ffffff
- ✅ 主色改用深蓝 #1a5276 或 墨绿 #2d6a4f（不要默认#007aff）
- ✅ 标题大号粗体 + 字间距收紧 letter-spacing:-0.5px
- ✅ 双层阴影叠加 0 1px 3px + 0 4px 24px
- ✅ 卡片不同布局（不全上下堆叠）
- ✅ 关键数字48-56rpx超大号
- ✅ 装饰元素：顶部细线、分界线、图标容器背景色
- ✅ 四周留白32-40rpx
- ✅ 选中/按下态有明确反馈（scale缩小+透明度变化）

## 在Claude Code prompt中使用
将此模板贴在prompt开头，再加具体设计任务。
