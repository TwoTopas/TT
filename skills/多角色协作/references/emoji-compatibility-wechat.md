# 微信小程序Emoji兼容性

## 核心原则
微信小程序的WebView使用系统字体渲染emoji。较新的emoji（Unicode 13.0+, 2020年后）在部分安卓设备上渲染为方框。

## 不兼容的emoji（避免使用）
| Emoji | Unicode | 名称 | 建议替换 |
|:-----:|:-------:|------|:--------:|
| 🧋 | U+1F9CB | 奶茶 | 🥤 (U+1F964, 杯装饮料) |
| 🫕 | U+1FAD5 | 火锅 | 🍲 (U+1F372, 锅) |
| 🫘 | U+1FAD8 | 豆类 | 用文字替代 |
| 🫗 | U+1FAD7 | 倒液体 | 用文字替代 |
| 🧉 | U+1F9C9 | 马黛茶 | ☕ (咖啡) |

## 微信内支持情况
- iOS微信: 支持较好，大部分Unicode 13.0+ emoji可渲染
- 安卓微信: 支持较差，Unicode 12.0+ 的部分emoji不渲染
- 开发者工具: 与真机表现不同，工具能看到的真机不一定能

## 最佳实践
1. 优先使用Unicode 10.0之前的emoji（2017年前）
2. 避免使用四肢/发色/性别组合emoji（如🧑‍🦰👩‍🦲）
3. 关键位置（品类卡片/按钮图标）使用通用emoji
4. 测试时在至少一台安卓真机上验证
