# Apple-Style Spreadsheet Template Design
## For taste-skill — Excel/Sheets templates that look premium

## Design System
- Font: Segoe UI (Windows SF Pro equivalent), size 11 for data, 18 bold for titles
- Title: 18pt bold, deep navy (#1D1D1F), 42px row height
- Spacer: 10px between title and headers
- Headers: 11pt bold, medium gray (#6E6E73), light gray background (#F5F5F7), 32px height
- Descriptions: 9.5pt, light gray (#86868B), same background as headers
- Data rows: 38px height (generous), Segoe UI 11pt, dark text (#1D1D1F)
- Borders: hairline #E5E5EA between rows only
- Status colors: Done=iOS green (#34C759), In Progress=iOS orange (#FF9500), Pending=iOS gray (#8E8E93)
- Auto-filter: enabled on header row
- Freeze panes: below header/description rows

## Apple Design Principles for Spreadsheets
1. Whitespace is the most important design element — 38px row height minimum
2. No background fills on data cells — white only
3. VERY subtle distinction between sections (light gray header bg #F5F5F7, nothing else)
4. Hairline borders only — no thick lines, no box borders around cells
5. Status colors use iOS system colors precisely
6. Column widths generous — minimum 18, scale with content up to 50
7. Landscape orientation, fit to width for printing
