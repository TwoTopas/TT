# 单案例深度分析提示词模板

> 标准化通用创业失败案例深度分析模板，适配餐饮、零售、美业、服务、电商、实体门店等绝大多数创业业态。
> 替换 `{{ }}` 标记的案例信息即可批量套用。

## 批量使用指南

1. 先将案例库所有条目整理成「待分析案例基础信息」表格格式
2. 逐条替换 `{{ }}` 占位符内容
3. 可根据行业特性在【角色设定】后补充特定行业要求

## Prompt 模板（供参考，不直接输入）

```
【角色设定】
请你以拥有10年以上创业风控咨询经验的商业数据分析师、实体创业运营专家的专业身份，
对下述创业失败案例进行全方位、结构化、数据化的深度复盘分析。
输出需兼顾专业性与可读性，核心结论直观可落地。

【核心输出规则】
1. 全程禁止大段纯文字堆砌，核心信息必须以「标准化表格 + 结构化数据可视化 + 高亮结论卡片」形式呈现
2. 所有分析必须紧扣案例真实数据，所有结论均需对应数据支撑，禁止空泛理论
3. 输出结构完全统一，便于批量归档进入案例库

【待分析案例基础信息】
```

## 适配稀疏数据字段

当案例数据不完整时，使用以下映射：

| 模板字段 | 可用数据源 | 处理方式 |
|---------|-----------|---------|
| 案例名称 | 视频标题 | 截取前30字 |
| 所属行业 | category字段 | 直接使用 |
| 所在城市 | 标题含城市名 | 正则提取，无则[数据不足] |
| 初始总投入 | amounts字段 | 取首个金额，转换单位 |
| 经营周期 | 标题含天数/月数 | 正则提取，无则[数据不足] |
| 核心失败原因 | reasons字段/标题推断 | 有则用，无则关键词匹配 |
| 其他字段 | 无 | 统一标注[数据不足] |
