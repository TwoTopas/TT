## 技术要点

### DeepSeek API调用参数
- model: deepseek-chat
- temperature: 0.2-0.3
- max_tokens: batch_size × 800 + 2000
- Retry: 3 attempts with 5-10s delay

### Path坑
- Python open() 需要 `D:/` 格式，不要用 `/d/`
- PowerShell `cd C:...` 可以，但Python路径写法不同

### 背景进程超时
- 25批 × 40s ≈ 17分钟
- 需要 background + notify 或 分拆执行
