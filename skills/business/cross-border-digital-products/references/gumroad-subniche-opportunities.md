## Current Trending Formats (TikTok/Reddit Verified)

These are what Gumroad creators are actively pushing in mid-2026:

1. **Mini Systems** 🥇 — Bite-size templates that do one thing fast (plan content, track goals, organize clients). Low price, crazy high demand.
2. **AI Prompt Packs** — Ready-to-use prompt libraries for specific industries/tools
3. **Digital Dashboards** — Plug-and-play planners in Notion or Canva for small business owners
4. **Swipe Files** — "Not boring ebooks." Collections of spicy hot takes, niche guides, inspiration files
5. **Notion + Sheets Bundles** — Same product available in both formats = higher price ceiling ($39-59)
6. **Digital Bundles** — Multiple templates packed together ($39-59 range outperforms individual items)

## What NOT to Make

| Product type | Why not |
|-------------|---------|
| Generic "500 AI Prompts" | Over-saturated; differentiate by vertical or not at all |
| Cheap $5-10 templates | Under $10 = 35% of listings, 0.8% of revenue |
| Graphic design assets | 38K products competing — unless extremely specialized |
| 3D assets | 2K+ products, high skill barrier |
| Yet another Notion dashboard | Notion template space is getting crowded |
| Anything you wouldn't buy yourself | If you can't see its value, neither will buyers |

## Decision Framework

When a user asks "what should I make?", run this filter:

```python
# Pseudocode for opportunity scoring
score = 0
if category_product_count < 1000:      score += 3  # Low competition
elif category_product_count < 2000:    score += 1
if revenue_per_product > 10000:        score += 2  # Good earnings
if success_rate > 40:                  score += 2  # Graveyard data
if can_build_in_1_to_3_days:           score += 2  # Fast validation
if price_can_be_29_to_49:              score += 1  # Sweet spot
if requires_coding:                    score -= 3  # Avoid for non-coders
```

Pick the top-scoring option and validate it with 3 independent data sources before building.
