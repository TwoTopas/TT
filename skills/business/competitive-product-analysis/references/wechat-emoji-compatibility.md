# 微信小程序Emoji兼容性（2026-06-23 session）

## 核心发现
微信小程序WebView使用系统字体渲染emoji。较新的Unicode emoji在部分设备上不显示（显示为方框□）。

## 不兼容的常见emoji

| Emoji | Unicode | 加入年份 | 替代方案 |
|-------|---------|---------|---------|
| 🧋 奶茶 | U+1F9CB | 2020 (13.0) | 🥤 杯装饮料 U+1F964 |
| 🫕 火锅 | U+1FAD5 | 2020 (13.0) | 🍲 锅 U+1F372 |
| 🫘 豆子 | U+1FAD8 | 2020 (13.0) | 用文字替代 |
| 🫓 馕 | U+1FAD3 | 2020 (13.0) | 🥟 或文字 |
| 🫒 橄榄 | U+1FAD2 | 2020 (13.0) | 文字替代 |

## 修复方式
```javascript
// ❌ 不兼容
{name:'奶茶', icon:'🧋', ...}

// ✅ 兼容
{name:'奶茶', icon:'🥤', ...}
```

## 测试方法
在微信开发者工具中编译后，查看品类卡片图标是否正常显示。如果显示为方框，说明该emoji不兼容。
