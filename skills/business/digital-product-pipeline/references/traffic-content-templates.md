# Traffic Content Templates

Reddit 帖子和 Twitter/X thread 的写作模板，用于 Gumroad 产品推广。

## Reddit 数据帖结构

### 标题格式

```
I built [product type] for [audience]. Here is what I learned about [topic].
```

或：

```
[Specific pain point] — what nobody tells you about [topic].
```

### 正文结构

```
1. 第一段：个人叙事切入（"I've been running community servers for years..."）
2. 2-3条硬知识/数据点，每条独立短段
3. "Here is the thing..." 转折段
4. 产品链接（"I ended up packaging this into a playbook..."）
5. 评论区互动钩子（"Anyone else struggle with X?"）
```

### 核心规则

- 第一句直接说做了什么，不铺垫
- 每段只说一个观点
- "Honestly I expected... but..." 自我纠正模式
- 结尾要抛一个具体问题

## Twitter/X Thread 结构

### 5-6条推文

```
Tweet 1: 钩子 — "I spent X months doing Y. Here is what I learned."
Tweet 2-3: 问题展开 + 具体场景
Tweet 4: 解决方案（说功能，不说产品名）
Tweet 5-6: CTA（链接 + 价值）
```

### 每条推文规则

- 自包含（不依赖上下文）
- 口语化，不用营销腔
- 具体数字 > 模糊描述
